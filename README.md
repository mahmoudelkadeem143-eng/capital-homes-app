# Capital Homes — نظام إدارة المشاريع

## الرفع على Vercel (أسهل طريقة)

1. روح [vercel.com](https://vercel.com) وسجل بحساب GitHub
2. اضغط **Add New Project**
3. اضغط **Upload** وارفع مجلد `capital-homes` كله
4. اضغط **Deploy**
5. هتاخد رابط جاهز تشاركه مع الفريق ✅

## تشغيل محلي

```bash
cd capital-homes
npm install
npm start
```
